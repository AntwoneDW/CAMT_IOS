Service Ports
These valves allow access
to the refrigerant for any
diagnostic or servicing
needs. The ports are
usually located on the
refrigeration lines near the
compressor, but can be
at different points in the
line set. Some ports can
also be used to isolate
one part of the refrigerant
system from another by
Service Ports
repositioning a valve stem.
Start and Run Capacitors
Electrical storage devices that help to overcome the torque
needed to start a motor, and almost all have run capacitors
that help motors run more efficiently. Run and start circuits
on compressor motors use capacitors that affect the
wattage, amperage draw, torque, speed, and efficiency of
the motor.
