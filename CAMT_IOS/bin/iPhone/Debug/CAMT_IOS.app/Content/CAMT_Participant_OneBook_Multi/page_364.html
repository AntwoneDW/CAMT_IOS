Check the Fan Motor Run Capacitor
A capacitor is an electrical storage device consisting of a layer of insulation sandwiched
between two metal plates. Here's how to check the capacitor for the fan motor.
Introduction
Run circuits on fan motors use capacitors that affect the wattage, amperage draw, torque, speed, and efficiency of the motor.
These capacitors help the fan motor run more efficiently.
Again, the most common problems affecting run capacitors are that they may be open—or that they may short out and fail.
You may be able to see evidence of the short if you observe bulging or leaking.
See "Checking the Start and Run Capacitors" on page 47 for more information on open or shorted capacitors.
If you find a capacitor to be defective, be sure to replace it with one specified by the manufacturer.
One More Time: Beware the Capacitor
Before you do any work on a fan motor, turn off all power to the unit and lock and tag out the circuit.
Then, be sure to discharge the run capacitor using a screwdriver, or the capacitor discharge tool.
If you fail to follow these critical safety steps, you could receive a severe shock.
